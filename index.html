    Sistema de Polling

    O Timberbot usa polling (sondagem) para verificar recompensas resgatadas a cada 2 segundos.

    Vantagem: Funciona sem necessidade de portas abertas ou IP público.

    Conexão com Timberborn

    Certifique-se de que:

    • Timberborn está rodando com o mod "HTTP Server" instalado
    • O servidor HTTP está escutando em localhost:8080
    • As alavancas estão criadas no mapa
